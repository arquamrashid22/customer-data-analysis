# 📊 Customer Data Analysis

## 🔍 Project Overview
This project focuses on analyzing a dataset of 1000 customers using Python. The goal is to explore the data, identify patterns, and generate meaningful insights through data analysis and visualization.

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- Matplotlib  

---

## 📂 Dataset
The dataset contains customer information such as:
- First Name  
- Last Name  
- Country  

---

## 📊 Analysis Performed
- Filtered data based on country (India-specific analysis)
- Identified top countries with highest number of customers
- Found most frequent first names
- Sorted and explored dataset for better understanding

---

## 📈 Visualizations
- Bar chart of top 5 countries by number of customers  
- Bar chart of top first names in India  

---

## 💡 Key Insights
- The country with the highest number of customers was identified dynamically using the dataset  
- Some first names appear multiple times, showing repetition patterns  
- Data distribution across countries is uneven  

---

## 🚀 Learning Outcomes
- Learned data filtering, sorting, and aggregation using Pandas  
- Practiced data visualization using Matplotlib  
- Improved understanding of real-world data analysis workflow
